Excel · openpyxl · WMA Forecasting
WMA Demand Forecasting Dashboard

Excel-based demand forecasting tool for 5 major steel customers (Tata Steel, JSW Steel, SAIL, Jindal Steel, Essar Steel), each with 30 POs of historical data. Uses a weighted moving average model (50/30/20 weights) to forecast both inter-arrival gap and order quantity per customer, with adjustable weights that trigger automatic recalculation across all sheets. Includes a summary dashboard pulling predicted next PO date, forecasted quantity, and MAE for all 5 customers into a single production planning view. Built entirely in Excel using openpyxl, no code required to operate.

06
Python · pandas · scikit-learn · matplotlib
Roll Wear & Change Optimization

ML-driven predictive maintenance system built on real Roll History Cards (CRM/F20) from Evonith Value Steel Ltd, Wardha. Digitized 56 campaigns across 3 rolls — 30,029 MT of actual production data. Key finding: in-house manufactured rolls averaged 739 MT/campaign vs 302 MT for competitor rolls (144% higher output) with a 33% vs 46% defect rate. Defect campaigns roll 58% less tonnage than normal campaigns. Includes a Random Forest classifier for defect prediction and a strategy simulation comparing fixed-schedule vs ML-guided roll change timing — ML-guided approach reduces quality escapes to zero.
